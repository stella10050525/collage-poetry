# 拼贴诗 · Collage Poetry 🦞

> 从对话的碎片中打捞诗意

一个 OpenClaw Skill，从任意对话上下文中提取文字片段，生成可交互的拼贴诗 HTML 页面。

## 效果

- **素材池**：对话中的文字被切割成 2-10 字的碎片，随机赋予不同字体、字号、颜色，模拟"从不同纸上剪下来的"效果
- **自由拼贴**：拖拽碎片到画布上，自由排列组合
- **AI 拼贴**：一键生成，交替使用 JS 随机组合和预设诗意排列两种模式
- **一键清空**：所有碎片回到素材池
- **导出作品**：将画布内容渲染为竖版 PNG 图片

## 视觉特色

| 特征 | 实现 |
|------|------|
| 剪切感边缘 | CSS `clip-path` 不规则多边形裁剪 |
| 胶带装饰 | CSS 伪元素模拟胶带粘贴效果 |
| 旧桌面背景 | CSS 渐变 + SVG 噪点纹理 |
| 碎片差异 | 随机分配 9 种中文字体、9 种墨色、8 种纸质底色 |
| 散落感 | 每个碎片有 -12° ~ +12° 随机旋转 |

## 作为 OpenClaw Skill 使用

将此仓库克隆到 OpenClaw 的 skills 目录下：

```bash
git clone https://github.com/<你的用户名>/collage-poetry.git
```

然后在对话中让我基于当前对话内容生成拼贴诗即可。

## 独立使用

直接打开 `templates/collage-poetry.html` 文件，浏览器即可运行（无需服务器）。

修改 `FRAGMENTS` 数组即可自定义素材池。

## 技术

纯 HTML + CSS + JavaScript，零依赖，零框架，零构建。
